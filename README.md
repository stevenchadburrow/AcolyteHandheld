# AcolyteHandheld

This is a PIC32MZ based game console.  It is programmed with my own NES emulator, as well as the <a href="https://github.com/deltabeard/Peanut-GB">Peanut-GB Gameboy Emulator</a>.  Most of the code is derived from my previous project <a href="https://github.com/stevenchadburrow/AcolyteHandPICd32">here</a>.
